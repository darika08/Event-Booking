Event Booking Website
Overview

The Event Booking Website is a simple frontend web application developed using HTML, CSS, and JavaScript. The project allows users to register, login, view available events, and book events online. User details and booking information are stored using the browser's Local Storage feature.

This project is designed for beginners to understand frontend web development concepts such as authentication, event handling, DOM manipulation, and local storage management.
